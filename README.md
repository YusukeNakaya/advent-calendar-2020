## どんな種類があるか

### 初級編

板を正面から観察してみましょう。

大きさが左右対象になっている、もしくはなっていないのがわかりますか？

#### ディレクショナル

ノーズ（前側）とテール（後側）の長さ、太さを比較したとき、ノーズの方が長く、太くなっています。

重心は板のやや後ろ側にあり、滑ると大きなノーズで雪を受け、浮力を得やすい形状です。

一方、逆方向には滑りにくい特徴があり、正方向にしか進まないことを前提とした遊び方に適しています。

普通にゲレンデを滑りたい人、スピードを出したい人、純粋に雪を楽しみたい人に向いていると言えます。

#### ツイン

ノーズとテールが全く同じ長さで、真ん中に重心があります。

正方向、逆方向ともに同じ性能を発揮でき、ターンやスピンをするときにも軸がわかりやすい形状です。

一方、ディレクショナルに比べるとスピードを出したときのバランスが難しく、またノーズが沈みやすいため浮力も得にくい特徴があり、それらを犠牲にして逆方向にもよく滑るような遊び方に適しています。

グラウンドトリックやジャンプなど、ダンスのように体を使ってゲレンデを楽しみたい人に向いていると言えます。

#### ディレクショナルツイン

ディレクショナルとツインのハイブリット型。

これには統一された定義がなく、各社独自に重心の位置やノーズとテールのバランスが異なります。

ただし確実に言えることは、完全なツイン（フルツイン）ではないということ。

つまり、逆方向に滑ったときに板の性能が落ちるので、いずれにしてもディレクショナルに近い乗り味です。

### 中級編

さて、今度は横から観察してみましょう。

微妙に湾曲しているのが見えますか？

#### キャンバー

もっともベーシックな形状で、ちょうど足の下あたりに接地点があります。

エッジが雪面にしっかり食い込み、ビシッとしたカービングができます。

また、その形状からしならせたときの反発が強く、ジャンプにも有効です。

雪面を捉える力が強い一方、コントロールミスすると意図しない食い込みで転びやすいです。

僕はこの現象を「エッジに怒られる」と言っています。一般的には「逆エッジ」と言います。

普通にゲレンデを滑りたい人、スピードを出したい人、純粋に雪を楽しみたい人に向いていると言えます。

また、グラウンドトリックやジャンプに慣れ、反発を使って遊びたい上級者にも向いています。

いろいろな形状を試して、結局キャンバーが一番良い、と言う人も多いです。

### ロッカー

板の中心付近に接地点が1箇所だけあります。

キャンバーに比べて雪面への食い込みが弱く、ターンではのんびりした乗り味です。

また、その形状からノーズが浮力を得やすいため、フワフワした浮遊感を味わうことができます。

雪面を捉える力が弱い一方、コントロール性が高く、また意図しない食い込みでも転びにくいです。

僕はこの現象を「なんでも許してくれる」と言っています。

グラウンドトリックやジャンプなど、ダンスのように体を使ってゲレンデを楽しみたい人に向いていると言えます。

### ダブルキャンバー

キャンバーとロッカーのハイブリッド型。

接地点が3箇所あり、波のような形状です。

ターンやジャンプなど、いろいろな遊び方を楽しみたい人に向いていると言えます。

### フラット

いっそのことキャンバーをなくした、ペタッとした形状です。

一見適当に作ったように見えますが、接地が点ではなく面になり、コントロール性が上がります。
